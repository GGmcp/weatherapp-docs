# WeatherApp Help

Welcome to the **WeatherApp** documentation. Use the sidebar to navigate the topics below.

{% hint style="info" %}
This app lets you log, view, and manage weather forecast records. You need an account to get started.
{% endhint %}

## What you can do

* Create and manage weather forecast records
* View temperature, humidity, and condition data
* Edit or delete existing forecasts
