# View Forecasts

After logging in you are taken to the **Weather Forecasts** page automatically.

The table shows all recorded forecasts with columns for:

| Column | Description |
|---|---|
| Date | The date of the forecast |
| Temperature | Temperature in °C |
| Summary | A short description (e.g. "Mild", "Hot") |

Use the **Details** link on any row to see the full record.
