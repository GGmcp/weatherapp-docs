# Add a Forecast

1. From the main forecast list, click **Create New**.
2. Fill in the fields:
   * **Date** — select the forecast date
   * **Temperature** — enter the temperature in °C
   * **Summary** — choose or type a short description
3. Click **Create** to save.

{% hint style="tip" %}
The summary field accepts free text, so you can enter custom descriptions like "Foggy morning".
{% endhint %}
